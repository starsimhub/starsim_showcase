# Starsim show and tell

A fortnightly showcase series for the Starsim community — developers and researchers sharing technical patterns, architectures, and workflows built on top of the framework.

Each session features short presentations on the code side of Starsim work: what classes were written, what patterns were developed, what was harder than expected. Think of it as a space for technical discussion about how we use and extend Starsim.
